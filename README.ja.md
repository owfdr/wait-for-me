# Wait For Me

自分だけのコレクションを作成して、楽しみましょう。

<img src="./assets/cover-image.ja.png" alt="Cover Image" width="512">

🧱 _このプロジェクトはまだ開発途中です。アプリが大幅に変更される可能性があります。_

## 機能

- 自分だけの個人コレクションを作成
- リンクやカスタム画像を追加
- 映画やドラマのウォッチリストを作成

その他にも多数！
