# Wait For Me

专属于你的个人收藏。

<img src="./assets/cover-image.zh.png" alt="Cover Image" width="512">

🧱 _此项目仍在大规模开发中。应用程序可能会经历很多重大变化。_

## 功能

- 打造自己的个人收藏
- 添加链接和自定义图片
- 为您喜欢的节目创建观看列表

还有更多！
