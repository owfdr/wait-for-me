# Wait For Me

당신의 개인 컬렉션을 만들고 관리하세요.

<img src="./assets/cover-image.ko.png" alt="Cover Image" width="512">

🧱 _이 프로젝트는 아직 큰 개발 중에 있습니다. 앱은 많은 중요한 변화를 겪을 수 있습니다._

## 기능

- 자신만의 개인 컬렉션 만들기
- 링크와 사용자 정의 이미지 추가
- 좋아하는 쇼를 위한 시청 목록 생성
